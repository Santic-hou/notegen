# 一、三亚目的地婚礼

保存了

要明确父母和自己在三亚的时间

共5天4晚，基本流程如下：

#### Day1：抵达・安顿・预热

- 上午：抵达三亚凤凰机场，专车接至酒店入住
- 下午：酒店休整，熟悉环境，新人试妆 / 试纱
- 晚上：三亚湾椰梦长廊散步，第一市场海鲜晚餐，熟悉环境携程

#### Day2：拍摄・彩排・熟悉

- 上午：婚纱照拍摄（亚龙湾 / 蜈支洲岛）
- 下午：婚礼流程彩排，与四大金刚确认细节
- 晚上：酒店附近享用海南特色烧烤，提前休息

#### Day3：婚礼・庆典・纪念

- 上午：新人化妆造型，布置场地
- 中午：轻松午餐，准备仪式
- 下午：**沙滩 / 草坪婚礼仪式**（1 小时），亲友合影
- 晚上：婚礼晚宴，小型庆祝派对

#### Day4：休闲・旅游

- 上午：自由活动，可选择潜水、冲浪等水上项目，或前往亚龙湾森林公园休闲放松
- 下午：根据新人喜好安排购物或参观景点，如南山文化旅游区
- 晚上：返程前的最后晚餐，与亲友共度温馨时

#### Day5：休闲・返程

优点：带老人出游，自己也完成一次蜜月出行；适合i人；更有氛围感；节省新人时间

缺点：提前预订酒店和景点门票，确保行程顺利且有足够时间休息；对老人比较折腾；出门开销多；不可控因素多（比如天气、婚庆布置等环节）

# 二、襄城县传统婚礼

优点：时间宽松，不必卡在4-5月份；家人亲人参与感强，日后话题少；老人不必在路途上折腾

缺点：缺点是人员多且杂，需要提前安排好车辆和住宿；婚礼场地选择和布置可能需要更多时间和精力投入
