# 三亚目的地婚礼流程

## 一、目的地婚礼与传统婚礼的主要区别


| 项目     | 传统婚礼       | 目的地婚礼           |
| -------- | -------------- | -------------------- |
| 地点     | 本地举办       | 外地/旅游景点举办    |
| 亲友参与 | 多本地亲友参与 | 主要是双方父母见证   |
| 仪式形式 | 通常较为正式   | 更加轻松、个性化     |
|          |                |                      |
| 旅行体验 | 没有           | 有（结合婚礼与旅行） |

## 二、三亚目的地婚礼的特点

目的地婚礼通常在风景优美、氛围浪漫的旅游胜地举行，如三亚的海滩或度假酒店，为新人和宾客提供独特的体验和难忘的回忆。

## 三、婚礼当天的流程

全天婚礼流程

<br/><br/><br/>6:30团队到酒店化妆<br/><br/>

8:30-9: 30第一套拍摄和录制晨袍（酒店布置好房间、窗花、囍字、贴纸、红色床品四件套）

<br/><br/>9:30-10:30 换服装以及化妆造型<br/><br/>

10: 30-12:30 第二套拍摄和录制<br/>

<br/>12: 30-13:30 休息➕午餐

<br/><br/>13: 30-14:30 换服装以及化妆造型<br/><br/>

14: 30-16:00第三套拍摄和录制视频

<br/><br/>16:00-17:00 换服装以及化妆造型

<br/><br/>17:00-18:30 第四套服装拍摄和录制海边加夜景烟花

<br/><br/>19:00-20:30团队返回门店

具体的拍摄流程细节摄影摄像与现场客人再沟通

仪式的具体时间可设置在10：30－12：30或14：30－16：00
